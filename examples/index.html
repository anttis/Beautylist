Beautylist Examples

Transforms a text input (or a textarea) into a nicer-looking input. For more info check out https://github.com/anttis/beautylist.

Basic

  $('#input-1').beautyList({
    dynamicInputSizing: true
  })

With a validator

  $('#input-2').beautyList({
    validators: [
      function(isA) { return isA === 'a'}
    ]
  })

Allowing spaces

  $('#input-3').beautyList({
    separator: /(?:,)/g,
    saveOnSpace: false,
  })

With a placeholder text

  $('#input-4').beautyList({
    placeholderText: 'Input here'
  })

With autocompletion

  $('#input-5').beautyList({
    autocomplete: {source: AUTOCOMPLETE_SOURCE}
  })
